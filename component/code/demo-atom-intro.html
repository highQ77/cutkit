class <!-- 就是 html 標籤 class 屬性 -->
style <!-- 就是 html 標籤 style 屬性 -->
repeat <!-- 就是元件重複個數，適合製作假資料列表 -->
event <!-- 綁定事件用於 prototype 互動程式部分，下方將有 event 解釋 -->
